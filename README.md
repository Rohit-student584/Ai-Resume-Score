# AI Resume Analyzer

An AI-powered Resume Analyzer built using the MERN Stack that helps job seekers analyze resumes against job descriptions and improve ATS scores.

## Features

- Google Authentication (Firebase)
- Resume Upload (PDF)
- AI Resume Analysis
- ATS Score
- Resume Match Score
- Strengths & Weaknesses
- Missing & Matched Keywords
- Resume Suggestions
- Resume History
- Resume Comparison
- Analytics Dashboard
- AWS Deployment

## Tech Stack

Frontend:
- React.js
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Firebase Admin SDK
- Multer

AI
- Google Gemini API

Deployment
- AWS EC2
- Nginx
- PM2
